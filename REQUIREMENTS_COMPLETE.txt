#                        INSTALLATION COMMANDS
# ============================================================================

# Installation Steps:
# 1. Create virtual environment: python -m venv peerlearn_env
# 2. Activate environment: source peerlearn_env/bin/activate (Linux/Mac) or peerlearn_env\Scripts\activate (Windows)
# 3. Install dependencies: pip install -r REQUIREMENTS_COMPLETE.txt
# 4. Install system dependencies: 
#    - Redis: sudo apt-get install redis-server (Ubuntu) or brew install redis (Mac)
#    - PostgreSQL: sudo apt-get install postgresql postgresql-contrib (Ubuntu)
#    - FFmpeg: sudo apt-get install ffmpeg (Ubuntu) or brew install ffmpeg (Mac)
# 5. Setup database: python manage.py migrate
# 6. Create superuser: python manage.py createsuperuser
# 7. Run development server: python manage.py runserver 0.0.0.0:5000
